## 💡 What is FlowMind?

FlowMind is not a to-do list. It's an **AI Execution Coach** that:
- Analyzes your workload in natural language
- Calculates a real-time **Confidence Score**
- Detects risk before deadlines are missed
- Generates autonomous recovery plans when you fall behind

---

## 🚀 Key Features

- 🧠 **AI Plan Generation** — type tasks naturally, get a structured execution plan instantly
- 📊 **Confidence Score** — 0-100 probability of meeting all deadlines
- 🚨 **Crisis Mode** — fullscreen intervention when score drops below 40
- 🆘 **Save My Day** — emergency AI triage: Strictly Do Today / Push Tomorrow / Drop Cancel
- ✅ **Auto Replan** — AI recalculates every time a task is completed
- 🎤 **Voice Input** — speak your tasks, AI generates the plan
- 🌅 **Morning Briefing** — auto-generated daily plan on first app open
- 📈 **AI Report Card** — productivity grade A-D with personalized feedback
- 🔮 **AI Prediction Engine** — future deadline risk forecasting
- 🔥 **Firebase Firestore** — real-time data persistence across sessions

## ⚙️ Local Setup

```bash
git clone https://github.com/ayushkumar1316/flowmind-ai-
cd flowmind-ai-
npm install
```

Create `.env.local` file:
```
VITE_GEMINI_API_KEY=your_gemini_api_key_here
VITE_FIREBASE_API_KEY=your_firebase_api_key_here
```

```bash
npm run dev
```
